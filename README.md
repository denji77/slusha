# Maidass

My simple telegram chat bot

## Requirements

- deno
- AI on localhost:3000/api/ (like this one https://github.com/sleroq/uapi)

## Preparation

1. `cp scripts/env.bash.example scripts/env.bash`
2. `chmod +x scripts/env.bash`
3. set up environment variables

## Start:

```bash
scripts/run.bash
```
